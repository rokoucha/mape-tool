# mape-tool

nftablesとiproute2を使ってMAP-E CEを動かすツール

## How to use

```
Usage:
  mape-tool [command]

Available Commands:
  apply       Apply MAP-E tunnel configuration
  calc        Calculate MAP-E parameters without applying
  completion  Generate the autocompletion script for the specified shell
  help        Help about any command

Flags:
  -h, --help   help for mape-tool
```
